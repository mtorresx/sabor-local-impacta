# 🎨 Design

Esta pasta contém todos os protótipos visuais, fluxos de navegação e o design system do SLFood, desenvolvidos no Figma durante a fase de design, anteriormente ao desenvolvimento em FlutterFlow.

## 📋 Descrição dos Módulos

### 📱 AppCliente
Interface do aplicativo para clientes finais
- **tela_login.png** — Autenticação e acesso ao app
- **FluxoCliente.png** — Jornada completa do usuário
- **ConfiguracaoCliente.png** — Perfil e preferências do cliente
- **design_system.png** — Componentes, cores, tipografia e grid

### 🏢 PainelGestao
Painel administrativo para gestores do restaurante
- **DashboardGestao.png** — Visão geral do negócio
- **ControleFinanceiro_Gestao.png** — Fluxo financeiro e relatórios
- **EstoqueGestao.png** — Gerenciamento de produtos
- **PromocoesGestao.png** — Campanhas e descontos
- **ConfiguracoesGestao.png** — Configurações da operação

### 👨‍💼 PainelFuncionario
Interface para colaboradores do restaurante
- **DashboardFuncionario.png** — Resumo de tarefas e pedidos
- **CardapioFuncionario.png** — Gestão de itens do cardápio
- **SuporteCliente_Funcionario.png** — Atendimento ao cliente

## 📌 Observações

- Todos os designs foram prototipados no **Figma** antes da implementação
- As telas representam a versão final aprovada para desenvolvimento
- Para acompanhar o progresso da implementação, confira as demais pastas do repositório
- Atualizações no design devem ser sincronizadas com o Figma antes de alterações no código
