# 🎨 Design

Esta pasta reúne as telas e protótipos criados no Figma durante a etapa de design do SLFood, antes do desenvolvimento no FlutterFlow.

## Conteúdo

- `telas-app-cliente/` — protótipos das telas do aplicativo do cliente (cadastro, login, cardápio, carrinho, pagamento).
- `telas-painel-admin/` — protótipos das telas do painel administrativo (gestão de cardápio, funcionários, pedidos).

## Observação

As imagens aqui representam a etapa de prototipação visual do projeto, anterior à implementação. Para ver as telas finais já implementadas, consulte as pastas `app-cliente/` e `painel-admin/` na raiz do repositório.
